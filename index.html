For cloning from the online repository and editing locally 1. git clone "ssh link"

2. git add . 3. git status **After this, modifications within the files will be tracked 4. git commit -m "Title Message" -m "Description"

5. git push origin main (or master; depending on the folder)

6. check for errors/git hub

For updating local code, 1. "git add ." the locally cloned repository 2. "git commit -m "some message"" 3. "git pull" the locally cloned repository
